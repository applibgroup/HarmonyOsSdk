◆ ConvertToString()

const char* ConvertToString ( const Format *  format)

Converts a media format object to a string for stream output or comparison.

Parameters
formatIndicates the pointer to the media format object created via Create().
Returns
Returns a string in [key1: valueType1(value1), key2: valueType2(value2)] format, where key indicates a metadata key. The valueType can be int32, float, string, or other. The value is the configured format data.
Note that the format data is data for a buffer metadata. For example: [max-height: int64(3000), mimetype: string(MPGE4), color-format: data]. SystemCapability.Multimedia.Media

void* GetBufferValue ( const Format *  format,

int32_t *  err,
size_t *  bufferSize 
)

Obtains the buffer information stored in this media format object.

This function returns a void pointer to the target buffer. The buffer size is saved in bufferSize.

Parameters
formatIndicates the pointer to the media format object created via Create().
formatTypeIndicates the pointer to the predefined metadata key matching the buffer information stored in this media format object.
errIndicates the pointer to the execution result. Set the value to 1.
bufferSizeIndicates the pointer to the buffer size.
Returns
Returns the void pointer to the buffer if the buffer information is stored in the media format object; returns a null pointer if an error or exception occurs. SystemCapability.Multimedia.Media
